# ESP32 mode STA

## Usage:

Create a `config.h` file, and edit it to look like this:

```h
#ifndef CONFIG_H
#define CONFIG_H

// Define your WiFi credentials here
#define WIFI_SSID "YourNetworkName"
#define WIFI_PASSWORD "YourPassword"

#endif // CONFIG_H
```
